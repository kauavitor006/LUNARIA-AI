# 🌙 LUNARIA AI

> **LUNARIA AI** é uma assistente virtual para desktop desenvolvida em Python, atualmente em fase **Alpha**, com foco em criar uma experiência moderna, modular e personalizável para interação com inteligência artificial.

O projeto utiliza o **OpenCode** como motor principal de IA e possui uma interface própria, desenvolvida para servir como base para futuras funcionalidades como automação, gerenciamento de memória, voz, plugins e integração com serviços externos.

## ⚠️ Status do Projeto

**Versão:** Alpha

A LUNARIA ainda está em desenvolvimento ativo. Muitas funcionalidades estão em evolução e algumas podem estar incompletas ou sofrer alterações entre as versões.

## Recursos disponíveis

* 💬 Interface gráfica para interação com IA
* 🧠 Integração com o motor OpenCode
* 🪟 Interface desktop desenvolvida em Python
* ⚙️ Estrutura modular para expansão do sistema
* 🎨 Tela de inicialização personalizada
* 🔌 Base para sistema de plugins
* 💾 Estrutura para memória e configurações
* 🔄 Arquitetura preparada para futuras atualizações

## Objetivo

A LUNARIA nasceu como um projeto experimental com o objetivo de evoluir para uma assistente pessoal completa, combinando inteligência artificial, automação e uma interface intuitiva em um único aplicativo.

O foco atual é construir uma base sólida antes da implementação de recursos mais avançados.
